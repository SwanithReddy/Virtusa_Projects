Welcome to this repository!
This repo contains three complete projects built using different technologies—Python, Java, and SQL. These projects are designed to demonstrate problem-solving skills, clean coding practices, and understanding of real-world applications.

1) Fare Calculator (Python)

Overview

A command-line based Python application that calculates travel fare based on distance, vehicle type, and time.


Features

* Accepts distance (in km)
* Supports different vehicle types (e.g., car, SUV)
* Applies base fare + per km rate
* Includes surge pricing based on time
* Displays final fare clearly


Tech Stack
Python

How to Run

python app.py



2) FinSafe Transaction Validator (Java)

Overview

A simple banking system that validates transactions such as deposits and withdrawals while ensuring proper balance management.

Features

* Create account with initial balance
* Deposit money
* Withdraw money with validation
* Prevent invalid operations (negative deposit, overdraft)
* Basic exception handling

Tech Stack
Java (OOP concepts)

How to Run

javac Main.java
java Main


3) Hospital Management System (SQL)

Overview

A relational database system to manage hospital data including patients, doctors, and appointments.

Features

* Store patient details
* Store doctor details
* Manage appointments
* Maintain relationships using foreign keys

Tech Stack
MySQL

How to Run

Create database Hospitalmanagement;
Run schema.sql;
Run queries.sql;
